## Stakeholder Roles

| Role | Definition | Power |
|------|------------|-------|
| **Economic Buyer (EB)** | Final authority, controls budget | Can say yes |
| **Champion** | Wants you to win, sells internally | Influence |
| **Coach** | Gives insider info | Intel only |
| **Technical Buyer** | Evaluates fit, runs POCs | Can veto |
| **User Buyer** | Will use daily | Can veto |
| **Influencer** | Has the EB's ear | Sway |
| **Blocker** | Against you | Can veto |
| **Ratifier** | Must approve (legal, security) | Can veto |

## Identifying Stakeholders

### Questions to Ask Your Champion

1. "Who else is involved in this decision?"
2. "Who could kill this deal?"
3. "Who controls the budget?"
4. "Who might have concerns?"
5. "Who does [EB] usually consult?"

### Signs by Role

**Economic Buyer signs**:
- Talks about budget without checking others
- Others defer to them
- Asks ROI questions, not feature questions

**Champion signs**:
- Proactively schedules meetings for you
- Shares internal docs
- Warns you about landmines

**Blocker signs**:
- Hard to schedule
- Asks "gotcha" questions
- Brings up past failures
- Delays without clear reasons
